# Nova

> AI Agents, Always On — Remote Cursor & Claude Code gateway for macOS.

Nova keeps your Cursor and Claude Code sessions running on a remote Mac, even after you close your browser.

## Download

**[⬇️ Download Nova 0.9.1-beta for Mac](https://nova.ironlab.cc/download/Nova-0.9.1-beta.dmg)**

Or visit [nova.ironlab.cc](https://nova.ironlab.cc)

## Install via Homebrew (coming soon)

```bash
brew install --cask ironlab-dev/nova/nova
```

## Requirements

- macOS 13+
- Cursor or VS Code installed
